## 🚀 ПЕРВЫЕ ШАГИ

```bash
# 1. Установка (2 мин)
git clone <repo>
cd compliance-audit
python3 -m venv venv
source venv/bin/activate
pip install selenium beautifulsoup4 requests langchain langchain-gigachat

# 2. Конфигурация (1 мин)
echo "GIGACHAT_API_KEY=your_token" > .env

# 3. Первый аудит (2 мин)
python -c "
from audit_engine import ComplianceAudit
audit = ComplianceAudit('https://example.com')
report = audit.run_audit(level2=False)
print(f'Score: {report.final_score}%')
"

# ✅ Готово! Дальше читай quick-start.md или tz-audit-v3-ru.md
```
