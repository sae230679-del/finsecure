## 🔑 JWT + BCRYPT

```javascript
// Регистрация
POST /auth/register
{
  "name": "Иван Петров",
  "phone": "+7 (999) 99-99-99",
  "email": "ivan@mail.ru"
}

// Backend:
1. Хешировать пароль (bcrypt, salt 10)
2. Сохранить в БД
3. Отправить email подтверждение

// Логин
POST /auth/login
{
  "email": "ivan@mail.ru",
  "password": "password123"
}

// Response:
{
  "access_token": "eyJhbGc...",
  "refresh_token": "refresh...",
  "user": { id, email, name, role }
}
```
